ІНСТРУКЦІЇ ЗІ ЗБИРАННЯ ТА ПОСІБНИК

1. ПРИЗНАЧЕННЯ
Цей документ містить технічну інформацію та детальні інструкції щодо компіляції консольного калькулятора. Файл знаходиться в папці /docs, що відповідає вимогам структурування проекту.

2. СТРУКТУРА ФАЙЛІВ
Проект організовано наступним чином:
- /src/calculator.cpp: Містить основний вихідний код програми на C++.
- /docs/manual.txt: Поточний посібник.
- /README.md: Загальний опис, мета проекту та інструкція із запуску.
- /.gitignore: Винятки для Git (бінарні та службові файли).

3. ДЕТАЛЬНА ІНСТРУКЦІЯ ЩОДО ЗБІРКУ (g++)
   Для компіляції потрібно мати встановлений компілятор g++.
   Виконайте такі команди в кореневому каталозі репозиторію:
   1. КОМПІЛЯЦІЯ:
      $g++ src/calculator.cpp -o app
   2. ЗАПУСК:
      $./app
