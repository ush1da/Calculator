ИНСТРУКЦИЯ И РУКОВОДСТВО ПО СБОРКЕ

1. НАЗНАЧЕНИЕ
   Этот документ содержит техническую информацию и подробные инструкции по компиляции консольного калькулятора. Файл находится в папке /docs, выполняя требование по структуризации проекта.

2. СТРУКТУРА ФАЙЛОВ
   Проект организован следующим образом:
   - /src/calculator.cpp: Содержит основной исходный код программы на C++.
   - /docs/manual.txt: Текущее руководство.
   - /README.md: Общее описание, цель проекта и инструкция по запуску.
   - /.gitignore: Исключения для Git (бинарные и служебные файлы).

3. ПОДРОБНАЯ ИНСТРУКЦИЯ ПО СБОРКЕ (g++)
   Для компиляции необходимо иметь установленный компилятор g++.
   
   Выполните следующие команды в корневом каталоге репозитория:
   
   1. КОМПИЛЯЦИЯ:
      $ g++ src/calculator.cpp -o app
      
   2. ЗАПУСК:
      $ ./app
